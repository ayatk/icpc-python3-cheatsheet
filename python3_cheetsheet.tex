\section{制御構造}

\subsection{三項演算子}
pythonでの三項演算子は
\begin{lstlisting}[language=Python]
  <条件式がTrueのときの値> if <条件式> else <Falseのときの値>
\end{lstlisting}

でできる。

\begin{lstlisting}[language=Python]
  >>> a = True
  >>> print('wei' if a else 'soiya')
  wei
\end{lstlisting}

\subsection{スワップ}

\begin{lstlisting}[language=Python]
  a, b = b, a
\end{lstlisting}

\section{文字列とか}

\subsection{0で桁あわせ}
0埋めの桁合わせをしたいときは一旦strに変換してから\texttt{zfill}メソッドで指定した
桁数分0埋めできる。
\begin{lstlisting}[language=Python]
  # 元のすうち
  >>> a = 12345
  # 合わせたい桁
  >>> l = 7
  >>> str(a).zfill(l)
  0012345
\end{lstlisting}
